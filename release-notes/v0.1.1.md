# Release 0.1.1 of the supersecret provider.

## What's New

- Published provider binaries, archives, and checksum files now use the provider name `terraform-provider-supersecret`.
- Private releases are now created as drafts first, so public sync only starts after the release is marked ready.
- The public release workflow continues to sync README, LICENSE, docs, examples, and release assets from the private source of truth.

## Notes

- No provider configuration changes are required for existing `l2fprod/supersecret` usage.
